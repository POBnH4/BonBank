In this report I discuss how the website works and give more details about it.
The purpose of this coursework is to have a web application that simulates a banking website, hence the name BonBank.
The functionality of the website is login/register/transfer money/ retrieve information/ delete user/ and logout.
The application is 3-tier based. The technologies used to create the web application above is JS/Java/DynamoDB/HTML/CSS/etc.

Used this website as an example for the homepage: https://www.barclays.co.uk/
Followed these instructions to set up tomcat:https://linuxize.com/post/how-to-install-tomcat-9-on-ubuntu-18-04/
